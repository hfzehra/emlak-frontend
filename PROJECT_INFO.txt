🚀 HEMEN BAŞLAYIN:

1. Migration Oluştur:
   cd Backend\Infrastructure
   dotnet ef migrations add InitialCreate --startup-project ..\API
   dotnet ef database update --startup-project ..\API

2. Test Verisi Ekle:
   SQL Server'da Backend\SeedData.sql çalıştır

3. Başlat:
   .\start.ps1
